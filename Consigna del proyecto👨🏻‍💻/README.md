# Sistema de Gestión de Contactos

Proyecto del módulo 2 desarrollado en Python.

## Descripción
Esta aplicación permite registrar, editar, eliminar y buscar contactos, almacenando información como nombre, teléfono, correo y dirección.

## Funcionalidades
- Agregar contactos
- Editar contactos
- Eliminar contactos
- Buscar por nombre
- Buscar por teléfono

## Estructura del proyecto
- `main.py`: menú principal e interacción con el usuario
- `contacto.py`: clase Contacto
- `gestor_contactos.py`: lógica de gestión de contactos
- `tests/test_gestor_contactos.py`: pruebas unitarias

## Requisitos
- Python 3

## Ejecución
En la terminal, ejecutar:

```bash
python main.py
